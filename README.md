# README #

Just download the source and run main.cpp to execute.

## What is this repository for? ##

### Quick summary ###
* A SNES Star Fox remake for COMP 371.
* Version 1.0

### Controls ###
* I -> invert Y-axis
* A/D -> left/right
* W/S -> up/down
* Q/E -> tilt left/right
* double tap Q/E -> do a barrel roll

### Debug ###
* G -> godmode
* L -> change lighting mode
* P -> cycle shaders
